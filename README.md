# AI Website Copy Generator for Local Businesses

## 📌 Project Overview
This project demonstrates how Prompt Engineering can be used to generate high-quality, conversion-focused website copy for local businesses.

The goal was to design a structured prompt system that produces:
- Homepage content with a clear value proposition
- Service page descriptions tailored to the business
- Persuasive call-to-action (CTA) sections
- Content that is ready to be used on a real website

## 🏢 Business Details

- **Type:** Café

### Positioning:
A cozy, aesthetic workspace café offering premium coffee and a calm environment for work, relaxation, and socializing.

---

## 🧠 Prompt Engineering Approach

This project uses a structured prompt framework with:

### 1. Role-Based Prompting
The AI is assigned a role (professional copywriter) to ensure high-quality, realistic output.

### 2. Context Injection
Clear business details are provided to avoid generic responses and improve relevance.

### 3. Structured Output Formatting
Each prompt enforces a defined structure (headline, subheadline, CTA, etc.) to make outputs usable directly on websites.

### 4. Tone Control
Prompts guide tone (friendly, calm, premium) to match the business identity.
